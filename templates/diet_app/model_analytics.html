Model Training Instructions

To generate or update the ML models and visualizations, run the training script:

python ml_training/train_models.py
What the training script does:
  • Loads and preprocesses datasets
  • Trains Random Forest models with optimized hyperparameters
  • Generates confusion matrices for classification
  • Creates learning curves showing model convergence
  • Produces feature importance rankings
  • Performs cross-validation
  • Saves trained models and visualizations

Technical Information

Technologies Used
  • scikit-learn: Machine learning algorithms
  • Random Forest: Ensemble learning method
  • Pandas & NumPy: Data manipulation
  • Matplotlib & Seaborn: Visualizations
